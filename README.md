
LSTM实现基于字的中文分词系统

lstm.py 定义了模型的结构

train.py 用于模型训练

lstm_decoder.py 加载并使用训练好的模型

分词示例：
输入：
讲述的是白清羽穷兵黩武后的四十年。东陆内部虚弱，皇室屡有皇嗣争夺和大臣擅权，外围诸侯自立。

输出：
['讲述', '的', '是', '白清羽', '穷兵', '黩武', '后', '的', '四十年', '。', '东陆', '内部', '虚弱', '，', 
'皇室', '屡', '有', '皇嗣', '争夺', '和', '大臣', '擅权', '，', '外围', '诸侯', '自立', '。']
